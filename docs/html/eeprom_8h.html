Detailed Description

Author
Braedon Giblin bgibl.nosp@m.in@i.nosp@m.astat.nosp@m.e.ed.nosp@m.u

This defines a basic interface for the TM4C MCU's onboard EEPROM. This supports basic read and write operations.

Note on addresses EEPROM consists of 32 blocks of 16 (64 bit) words. So, to make address 1 byte, EEPROM is paged into two pages (through software). The upper four bytes of an address consist of the block, and the lower 4 bytes represent the word.
