\doxysubsection{Detailed Description}
\begin{DoxyAuthor}{Author}
Braedon Giblin \href{mailto:bgiblin@iastate.edu}{\texttt{ bgiblin@iastate.\+edu}}
\end{DoxyAuthor}
This defines a basic interface for the T\+M4C M\+CU\textquotesingle{}s onboard E\+E\+P\+R\+OM. This supports basic read and write operations.

{\bfseries{Note on addresses}} E\+E\+P\+R\+OM consists of 32 blocks of 16 (64 bit) words. So, to make address 1 byte, E\+E\+P\+R\+OM is paged into two pages (through software). The upper four bytes of an address consist of the block, and the lower 4 bytes represent the word.
